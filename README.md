# TravelEase – Vibe Coding Django Project

TravelEase is a simple travel booking web application built using Django. It follows Vibe Coding principles by using AI assistance to generate, review, refine, and improve the project.

## Features

- View travel destinations
- Submit travel booking requests
- Store bookings in SQLite database
- Manage destinations and bookings from Django admin panel
- Simple HTML-based user interface

## Technologies Used

- Python
- Django
- HTML
- SQLite
- GitHub

## How to Run

```bash
git clone https://github.com/nirakarshrestha-web/Vibe-coding.git
cd Vibe-coding
pip install -r requirements.txt
py manage.py migrate
py manage.py runserver
